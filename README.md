✨ Features
📋 Copy All Tab URLs

Copy every open tab with a single click or use the global keyboard shortcut.

Perfect for sharing research, creating backups, documenting projects, or importing into other tools.

🚀 Paste & Open URLs

Paste a list of:
• URLs
• Search queries
• Mixed content

URLBlast automatically:
• Opens valid URLs in new tabs
• Converts non-URLs into Google searches
• Warns before opening a large number of tabs

Ideal for restoring browsing sessions or launching research in seconds.

🔍 Smart Tab List
Quickly find exactly the tab you're looking for.

Features include:
• Live search
• Select multiple tabs
• Copy individual URLs
• Export selected tabs
• Instantly switch to any open tab

No more hunting through dozens (or hundreds) of browser tabs.

💾 Save Sessions
Save your current browsing session with a custom name and restore it anytime.

• Save up to 20 sessions
• Restore with one click
• Perfect for research, work, study, and projects

Never lose an important set of tabs again.

🤖 AI Categorize (Free)
Powered by Google Gemini, URLBlast can intelligently organize your open tabs into meaningful categories such as:

• Development
• Shopping
• News
• Documentation
• Entertainment
• Learning
• Finance
• Productivity

You can instantly create Chrome Tab Groups from the AI-generated categories.

Privacy note: AI categorization only works when you provide your own Google Gemini API key. No API keys are stored remotely.

⌨️ Global Keyboard Shortcuts

Control URLBlast without opening the popup.

Shortcut	= Action
Alt + Y	  = Open URLBlast popup
Alt + K	  = Copy all tab URLs
Alt + V	  = Open URLs from clipboard

All shortcuts are fully customizable through Chrome's keyboard shortcut settings.

From Source
Clone this repository.
git clone https://github.com/yourusername/urlblast.git
Open Chrome and navigate to:
chrome://extensions
Enable Developer mode.
Click Load unpacked.
Select the project folder.
Permissions

URLBlast requests only the permissions required for its features:

tabs – Access open tabs
tabGroups – Create AI-generated tab groups
storage – Save settings and sessions locally
clipboardRead – Read clipboard for bulk opening
clipboardWrite – Copy exported URLs
commands – Global keyboard shortcuts

No browsing history is transmitted anywhere.

AI Categorization

To enable AI organization:

Generate a Google Gemini API key.
Open URLBlast Settings.
Paste your API key.
Click Categorize Tabs.

Your API key remains stored locally in your browser.
