🚀 CORE FEATURES
• Copy All Tab URLs — Copy every open tab's URL with one click (or use the global shortcut Alt+K). Choose from 7 export formats: URLs Only, Title + URL, Markdown, HTML, JSON, CSV, and Org-mode.
• Paste & Open URLs — Paste a list of URLs or search terms, and URLBlast opens them all in new tabs instantly. Non-URLs are automatically turned into Google searches (with a bulk-open safety warning).
• Smart Tab List — Browse, search, and select tabs in a clean, searchable list. Switch to any tab, copy individual URLs, or select a subset to export.
• Save Sessions — Save your current tabs as a named session and restore them later. Never lose your research tabs again. Supports up to 20 saved sessions.
• AI Categorize (Free) — Powered by Google Gemini, URLBlast can intelligently group your open tabs into categories like "Development", "Shopping", "News", etc. Create Chrome Tab Groups directly from AI results.
• Global Keyboard Shortcuts:
  - Alt+Y — Open URLBlast popup
  - Alt+K — Copy all tab URLs to clipboard (works even without opening the popup)
  - Alt+V — Open URLs from clipboard in new tabs (global shortcut)
  Shortcuts are fully customisable via browser settings.
🛡️ PRIVACY FIRST
URLBlast is 100% privacy-respecting. It collects zero data. No analytics, no telemetry, no remote servers (except when you explicitly use the AI feature with your own Google Gemini API key). All session and settings data is stored locally in your browser.
⚡ LIGHTWEIGHT & FAST
Built with Manifest V3, URLBlast is a lean service-worker-based extension. It starts instantly and never slows your browser down.
🌙 BEAUTIFUL DARK THEME
Features a premium dark UI with light mode support and system theme detection. Clean, modern design with smooth animations.
🔔 IN-EXTENSION NOTIFICATIONS
All activity notifications (copied URLs, opened tabs, donation reminders) are delivered in a clean built-in notification page — no intrusive OS desktop pop-ups.
📋 EXPORT FORMATS
Export your tabs in the format your workflow needs:
• URLs Only — Clean list of URLs
• Title + URL — Human-readable with page titles
• Markdown — [Title](URL) links
• HTML — Clickable hyperlinks
• JSON — Structured data for developers
• CSV — For spreadsheets
• Org-mode — For Emacs Org-mode users
